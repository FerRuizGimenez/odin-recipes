## Project: Recipes

### Overview
This project is part of the Foundations curriculum from The Odin Project. It involves the development of a static recipe website using only HTML.

The application consists of a main index page that serves as a navigation hub, linking to multiple individual recipe pages. Each page is structured to present content in a clear, semantic, and accessible manner.

### Technical Focus
The primary focus of this project is to reinforce core HTML concepts and best practices, including proper document structure and semantic markup.

### Skills Demonstrated
- Semantic HTML5 structure (headings, lists, links, images)
- Multi-page website architecture and internal navigation
- File and directory organization following standard conventions
- Separation of concerns at a structural level (content-focused HTML)
- Use of relative paths for linking resources
- Basic accessibility considerations through proper HTML semantics
